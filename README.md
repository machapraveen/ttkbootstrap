## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher



### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/machapraveen/ttkbootstrap.git
   cd ttkbootstrap
   ```

2. **Install dependencies**
   ```bash
   pip install -e .
   
   ```

3. **Run the application**
   ```bash
   python main.py
   ```
